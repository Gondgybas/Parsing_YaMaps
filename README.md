# 🗺️ Parsing YaMaps — Парсер контактов с Яндекс.Карт

Десктопное приложение с графическим интерфейсом для автоматического сбора контактной информации компаний (email, телефон, сайт, адрес, вид деятельности) из Яндекс.Карт.

![Python](https://img.shields.io/badge/Python-3.9+-blue?logo=python&logoColor=white)
![Selenium](https://img.shields.io/badge/Selenium-Chrome-green?logo=selenium)
![License](https://img.shields.io/badge/license-MIT-yellow)

---

## 📋 Возможности

- 🔍 Поиск компаний по произвольному запросу (например, `металлообработка Подольск`)
- 📧 Трёхуровневый поиск email:
  1. На карточке Яндекс.Карт
  2. На сайте компании (с обходом страниц `/contacts`, `/about` и др.)
  3. Через Яндекс.Поиск (top-3 результата)
- 📞 Сбор телефонов с Яндекс.Карт и сайтов
- 🏢 Парсинг адреса, вида деятельности, сайта
- 📊 Сохранение результатов в Excel (`contacts_database.xlsx`)
- 🔄 Защита от дублей (по триплету: название + адрес + сайт)
- ⏸️ Пауза / Продолжение / Остановка парсинга
- 📂 Встроенный просмотр базы данных
- 📝 Логирование в файл (папка `logs/`)
- 🖥️ Тёмный интерфейс на CustomTkinter

---

## 🖼️ Интерфейс

Приложение имеет тёмную тему с кнопками управления парсингом, полем ввода запроса, ограничением количества компаний и окном отладочных сообщений в реальном времени.

## ⚙️ Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Gondgybas/Parsing_YaMaps.git
cd Parsing_YaMaps
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

Или вручную:

```bash
pip install customtkinter pandas openpyxl selenium webdriver-manager beautifulsoup4 requests urllib3
```

### 3. Убедитесь, что установлен Google Chrome

Приложение использует Chrome через Selenium. [Скачать Chrome](https://www.google.com/chrome/)

---

## 🚀 Запуск

```bash
python parser.py
```

### Как пользоваться

1. Введите поисковый запрос (например, `автосервис Москва`)
2. (Опционально) укажите лимит компаний
3. Нажмите **«Начать парсинг»**
4. Дождитесь открытия браузера с Яндекс.Картами
5. **Прокрутите список компаний вручную до конца** (мышкой / колёсиком / PageDown)
6. Нажмите **OK** в появившемся диалоговом окне
7. Парсер начнёт обрабатывать карточки автоматически
8. Результаты сохраняются в `contacts_database.xlsx` после каждой компании

## 📦 Сборка в .exe (PyInstaller)

```bash
pyinstaller --onefile --noconsole --name "YaMapsParser" --icon=icon.ico ^
  --collect-all selenium ^
  --collect-all webdriver_manager ^
  --collect-all customtkinter ^
  --hidden-import=bs4 ^
  --hidden-import=requests ^
  --hidden-import=urllib3 ^
  --hidden-import=pandas ^
  --hidden-import=openpyxl ^
  parser.py
```

> **Важно:** положите `chromedriver.exe` в ту же папку, что и собранный `.exe` — это страховка на случай, если `webdriver-manager` не сможет скачать драйвер автоматически.

Готовый файл появится в папке `dist/`.

## ⚠️ Ограничения

- Яндекс может показать **капчу** — её нужно решить вручную в окне браузера
- Список компаний нужно **прокрутить вручную** перед парсингом
- Чёрный список доменов фильтрует агрегаторы (Avito, 2GIS, VK и др.)
- Работает только с Google Chrome
